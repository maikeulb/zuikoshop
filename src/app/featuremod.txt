module     declarations   providers    exports
Domain ->  yes            rare         top component
Routed ->  yes            rare         no
Routing -> no             yes(guards)  routingmodule
Service -> no             yes          no
Widget ->  yes            rare         yes

e.g
usermodule imported by the coremodule
uimodule imported by the sharedmodule

module     can be importedby     examples
domain     feature, app          contactmodule(before routing)
routed     nobody                contactmodule, dashboard
routing    feature (forrouting)  approuting, contactrouting
service    app                   http, core
widget     feature               common, shared

modules to import each time:
commonmodule
formsmodule
routermodule(without forroot() or forchild())
any other module giving components, directives, or pipes

modules to import only once:
httpclient
browseranimations
any other module provinding you servies only
